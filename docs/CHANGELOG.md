# SuperOne 版本说明

## 🎯 产品简介

豆流便签（SuperOne）是一款集成多种实用工具的 uTools 插件，帮助你在 PC 端和手机端高效管理日常工作与思考。

**核心功能模块**：
- 📝 **主题管理**：使用 Markdown 记录思考与知识，支持时间线日志管理
- ✅ **检查清单**：创建和管理检查清单，支持执行记录和进度跟踪
- 📋 **任务计划**：管理近期任务，支持任务创建、编辑、状态流转和删除
- 📈 **投资标的**：管理股票信息，记录和跟踪投资思考

**核心特性**：
- 🔒 字段级数据加密，数据库无明文存储，保障数据安全
- 🔄 跨平台同步，uTools + 微信小程序无缝互通，一个账号全平台使用
- 🔐 微信登录，无需注册，支持多设备同时使用

---

## 🔒 核心特性

### 1. 字段级数据加密
- ✅ 每个敏感字段独立加密，数据库实现无明文存储
- ✅ 即使数据库泄露，也无法还原你的原始内容

### 2. 跨平台无障碍互通
- ✅ uTools + 微信小程序，一个账号全平台同步
- ✅ PC 端编辑，手机端立即可见；手机端查看，PC 端同步更新
- ✅ 无需注册，直接与微信绑定，支持多设备同时使用

---

## 🚀 快速开始

1. 在 uTools 中搜索 "豆流便签" 安装插件
2. 微信扫码登录，自动同步数据
3. 创建主题，使用 Markdown 记录想法
4. 在微信小程序中搜索 "豆流便签"，随时查看和编辑
